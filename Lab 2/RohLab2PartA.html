Definitions and Descriptions

Table
- A table is a structured set of data in rows and columns. These are important because they are way of relaying and showing relationships between information.
tr
- The tr tag stands for table row and depicts a row in a table. The tr tag must be used in between the table tags.
th
- The th tag stands for table header and depicts a header cell in a table. The th tag must be used in between the table tags.
td
- The td tag stands for table data and depicts a cell in a table. The td tag must be used in between the table tags.
thead
- The thead tag is used to group the header content in a table. The thead tag must be used in between the table tags.
tfoot
- The tfoot tag is used to group the footer content in a table. The tfoot tag must be used in between the table tags.
Caption
- The caption tag is used to define a table caption. The caption tag must be used in between the table tags.
Colspan
- The colspan attribute allows you to extend the number of columns a cell is a part of.
Rowspan
- The rowspan attribute, as opposed to the colspan attribute, allows you to extend the number of rows a cell is a part of.
Cellspacing
- The cellspacing attribute allows you to manipulate the space between table cells.
Padding
- The padding property, as opposed to the cellspacing attribute, allows you to add space between the border and text of an element.
Table border
- The border property adds the border or outline of an element.
Border-width
- The border-width property allows you to manipulate the width or size of an element's outline. You can apply this all four sides of an element. To control border-widths, make sure you include a border-style property before it. Then, you can include the border-width tag and change the thickness of the border. For example: border-style: solid; border-width: thin;
